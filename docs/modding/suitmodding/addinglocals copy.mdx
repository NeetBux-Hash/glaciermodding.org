### **2. A LOCR?**
*Whats in a locr?*
*Inside the LOCR file is your text.* 

Lets visualise a Locr to understand the parts better

Simplified version of a LOCR file and explain what 
```json
     "en": {
        "string1": "text you see in game",
        "string2": "More text you see in game"
     },
     "fr": {
        "string1": "text tu vieux en jeu",
        "string2":"Plus de text tu vieux en jeu"
     }
     ```
     We can learn that:
     
    1. A <Tooltip text={<><strong>string</strong><span className="icon-alert"></span></>} tooltip= "A key that links to text in the LOCR File" /> is attached to each entry in the file
    2. Same **"strings"** are used in different arrays for `{multiple languages}`
    3. Every essential part is in `"quotes"`
     
     These are the rules of the files or the <Tooltip text={<><strong>syntax</strong><span className="icon-alert"></span></>} tooltip="A set of rules that define how code is written and organized." />
  > We will recreate this structure in the manifest to replace the text. Just how we did with blobsfolders 
  
- **Search Your Text**: Use `Ctrl + F` in the "Preview Box" to locate your text.
- **Copy the String**: Be sure to copy the string without quotes, as you’ll need it for the next steps.
<details>
<summary>String?</summary>

To further clarify what a string is. It's the long piece of code before the text you see in-game

    Example of a suit's name *line*:
 ```json
 "989928F2-06D6-42F3-871A-353F07DEF969_LEGACY_HERO_BLOODMONEYSUIT_M_HPA2293_NAME_": "Blood Money Suit"
 ```
 - The first part is the "string": `989928F2-06D6-42F3-871A-353F07DEF969_LEGACY_HERO_BLOODMONEYSUIT_M_HPA2293_NAME_`
 - The second part will be the text you see in game: `Blood Money Suit`
</details>

### **3. Replacing Your Text in the Manifest**
*Now, let's update the manifest:*
- **Open Your Manifest**: Navigate to your manifest file.
- Add this line: `"localisationOverrides": {}`
 :::tip 
Suggestions 
blbabka learn how to type code with suggestions
:::
- Add your LOCR HASH and a language array: 
```json
   "localisationOverrides": {
       "yourLOCRhash": {
           "english": {}
       }
   }
   ``` 
:::danger  
   The next logical step would be to add our string and new text.
 ```json
   "localisationOverrides": {
       "yourLOCRhash": {
           "english": {
            "string_you_copied": "Your new text"
           }
       }
   }
   ``` 
  > But we can't do that!
  
   We first need to convert the string to decimal (meaning containing numbers only) code. Then we can finish the manifest like you just saw.
:::
>We cant use string in manifest so go to:
- **Text Tools**: Open Text Tools and:
  - **Paste Your String**: Use the Localisation Hash Calculator in Text tools to convert the string.
  :::tip 
*Use Windows Clipboard (`Windows + V`) to copy and paste the codes youre copying.*
:::
  - **Copy the Decimal**: You'll use this in your manifest.
- **Add Arrays and New Text**: Update your manifest with arrays, pasting the new decimal and your custom text.
Result:
```json
"localisationOverrides": {
    "yourLOCRhash": {
        "english": {
            "yourconvertednamedecimal": "New Suit Name"
        }
    }
}
````
>This recreates syntax like locr (remember simpliefied locr?)

### 5. (Optionally) Add more languages
#### Same Strings:
In [Part 2](#2-copy-the-string) of the tutorial we demonstrated that *strings* are the same in each <Tooltip text={<><strong>language array</strong><span className="icon-alert"></span></>} tooltip="A single language array in a LOCR file consisting of strings" />
>*This will make adding languages for each entry very easy.*
    

To have your mod support more languages, simply:
- make another <Tooltip text={<strong>language bracket</strong>} tooltip="A single language array that can contain multiple entries of strings each attached to text you see in-game" /> 
- add the <Tooltip text={<strong>same decimal</strong>} tooltip="A string converted into a numeral form" /> with your text


Example of a new language line:
<div id="langcode"></div>
```json
"localisationOverrides": {
        "yourLocrHASH": {
            "english": {
                "yourconverteddecimal": "New Suit Name"
            },
            "french": {
                "yourconverteddecimal": "Niveua Costume nome"
            }
        }
    }
```

### Adding More LOCR Files

*If you need to modify more* <Tooltip text={<strong>LOCR Files</strong>} tooltip="A file that holds game text in different languages, linked by unique 'strings'." />: 

1. **Insert a Comma**: After your previous <Tooltip text={<strong>LOCR Array</strong>} tooltip="an array in the manifest resembling the game's LOCR file syntax" />  in the <Tooltip text={<strong>manifest</strong>} tooltip="A file in your mod folder that holds important information and can execute code" />.
2. **Repeat [the steps from start](#1-identify-your-locr)** (or continue here)
2. **Add a New** <Tooltip text={<strong>LOCR Array</strong>} tooltip="an array in the manifest resembling the game's LOCR file syntax" />: Use the same structure, making sure to follow the <Tooltip text={<strong>syntax rules</strong>} tooltip="A set of rules that define how code is written and organized." />.
3. <Tooltip text={<strong>Repeat the Conversion</strong>} tooltip="Using the Localisation Hash calculator" />: Use <Tooltip text={<strong>Text Tools</strong>} tooltip="A tab in GlacierKit that provides tools for converting text and hashes for game modifications." /> again to convert any new <Tooltip text={<strong>strings</strong>} tooltip= "A key that links to text in the LOCR File" /> into <Tooltip text={<strong>decimals</strong>} tooltip="A string converted into a numeral form" />.
4. **Update the** <Tooltip text={<strong>manifest</strong>} tooltip="A file in your mod folder that holds important information and can execute code" />: Add the new language entries, keeping everything consistent.

Example of every aspect from this tutorial applied:
```json
"localisationOverrides": {
    "yourLocrHASH": {
        "english": {
            "yourconverteddecimal": "New Suit Name",
            "yourotherconverteddecimal": "New Suit Description/or Other New Suit name"
        }
    },
    "otherLocrHASH": {
        "english": {
            "somenewconverteddecimal": "More New Suit Name",
            "someotherconverteddecimal": "More New Suit Description/or Other More New Suit name"
        },
        "spanish": {
            "somenewconverteddecimal": "Más nombre de traje nuevo",
            "someotherconverteddecimal": "Más descripción del traje nuevo/u otro nombre del traje más nuevo"
        }
    }
}
```

*Remember to keep your* <Tooltip text={<strong>manifest</strong>} tooltip="A file in your mod folder that holds important information and can execute code" /> *clean and organized to avoid errors.*
